## Description

This project is part of the final work for the Higher Vocational Training cycle. It is a Node.js application developed using the NestJS framework and PostgreSQL database, designed to manage hotel room reservations.

## Installation

```bash
$ npm install
```

## ENDPOINTS

- /users
- /hotel
- /rooms
- /reservations
- /cars
- /flights
- /activities

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
